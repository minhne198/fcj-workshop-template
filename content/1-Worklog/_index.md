* I completed the AWS Cloud internship and learning program over **12 weeks** through the AWS Study Group hands-on lab series. Throughout the process, I progressively explored core AWS services, from basic infrastructure services to data analytics solutions and system administration on the cloud platform.

* In the first weeks, I focused on becoming familiar with the AWS environment, compute, storage, and networking services such as Amazon EC2, Amazon S3, Amazon VPC, and resource management tools. Next, I practiced deploying databases with Amazon RDS, learned about access management with AWS IAM, and applied security measures using AWS KMS, IAM Role, and AWS CloudTrail.

* In the next stage, I researched and practiced NoSQL services with Amazon DynamoDB, built automation solutions using AWS CLI, Lambda, and CloudWatch, and learned how to manage, monitor, and optimize resources on AWS.

* In the final weeks, I focused on Data Analytics on AWS. I built a Data Lake architecture using Amazon S3, processed data with AWS Glue, queried data with Amazon Athena, and visualized data with Amazon QuickSight. In addition, I practiced data preparation with AWS Glue DataBrew and built a Data Warehouse with Amazon Redshift to support analytics and reporting needs.

* Through 12 weeks of hands-on practice, I gained a clearer understanding of cloud computing architecture, deployment and operation processes on AWS, and improved my skills with the AWS Console, AWS CLI, and modern data services. This knowledge and experience form an important foundation for my future study, research, and career development in Cloud Computing and Data Engineering.
